# 🧠 MindScroll
### Transforming Scrolling into Learning

MindScroll is a micro-learning platform that converts passive scrolling into meaningful knowledge consumption.  
Instead of endless entertainment feeds, users discover **short knowledge cards, quizzes, news insights, and challenges** in a familiar vertical scrolling interface.

Built during **Hackverse 2.0 (24-Hour Hackathon)** organized by **HSNC University at Watumull Institute of Engineering & Technology**.

## 📌 Problem

People spend hours scrolling through content feeds, yet very little of that time contributes to real learning or knowledge growth.

---

## 💡 Solution

MindScroll turns scrolling into **micro-learning** by delivering:

• Knowledge cards  
• News summaries  
• Interactive quizzes  
• Actionable challenges  
• AI-powered explanations

The system encourages learning through **gamification (XP, streaks, and daily goals)**.

---

## ✨ Key Features

• Vertical **knowledge feed** optimized for mobile  
• **AI "Explain"** feature for deeper understanding  
• **Quizzes & challenges** for active learning  
• **XP system and learning streaks**  
• **Personalized interests onboarding**  
• **Dark / Light mode UI**  
• **English & Hindi language support**
